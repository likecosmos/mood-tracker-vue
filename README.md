# 🌈 心情记录器（Vue 3）

一个用于记录和统计每日心情的轻量级单页应用。

## ✨ 功能特性

- 😊 😐 😢 快速选择心情
- 💾 本地存储，刷新不丢失
- 🗑 单条删除 / 一键清空
- 📊 心情统计
- 🌗 暗黑模式
- 🔗 模拟接口请求（Axios）
- 🧠 Pinia 状态管理
- ⚡ Vite + Vue 3

## 📦 技术栈

- Vue 3
- Vue Router
- Pinia
- Axios
- Vite

## 🚀 快速启动

```bash
npm install
npm run dev
```

## 📸 效果预览

> 登录页 / 首页 / 暗黑模式

## 📌 项目亮点

- 组件化设计（MoodPicker / MoodList）
- 状态与 UI 解耦
- 贴近真实前端工程化实践
