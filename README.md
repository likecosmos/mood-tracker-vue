# 🌈 心情记录器（Vue 3）

一个用于记录和统计每日心情的轻量级单页应用。

## ✨ 功能特性

- 😊 😐 😢 快速选择心情
- 💾 本地存储，刷新不丢失
- 🗑 单条删除 / 一键清空
- 📊 心情统计
- 🌗 暗黑模式
- 🔗 模拟接口请求（Axios）
- 🧠 Pinia 状态管理
- ⚡ Vite + Vue 3

## 📦 技术栈

- Vue 3
- Vue Router
- Pinia
- Axios
- Vite

## 🚀 快速启动

```bash
npm install
npm run dev
```

## 📸 效果预览

> 登录页 / 首页 / 暗黑模式

## 📌 项目亮点

- 组件化设计（MoodPicker / MoodList）
- 状态与 UI 解耦
- 贴近真实前端工程化实践

简历写法：
心情记录器｜Vue 3 单页应用​
基于 Vite + Vue 3 构建，采用 Composition API 与组件化开发模式
使用 Vue Router 实现页面路由，Pinia 进行全局状态管理
封装 MoodPicker、MoodList 等可复用组件，降低耦合度
通过 localStorage 实现数据持久化，支持删除 / 清空与统计功能
接入 Axios 模拟接口请求，体验前后端协作流程
支持暗黑模式切换，结合 CSS 过渡提升用户体验
